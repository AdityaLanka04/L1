## üí° Tips & Tricks

### Working with Tables
1. **Quick Insert**: Use the table button dropdown for common sizes
2. **Modify Structure**: Right-click any cell to add/remove rows and columns
3. **Cell Selection**: Click and drag to select multiple cells
4. **Merge Cells**: Select multiple cells, right-click, and choose "Merge cells"
5. **Styling**: Tables automatically have alternating row colors and hover effects

### Efficient Note-Taking
1. **Use Folders**: Organize notes by subject or project

4. **AI Assistance**: Select text and use AI to improve or expand your ideas

1. **Headers**: Use H1 for main title, H2 for sections, H3 for subsections
2. **Lists**: Use checklists for tasks, bullets for items, numbers for steps
3. **Code Blocks**: Perfect for code snippets, commands, or technical content
4. **Blockquotes**: Great for highlighting important information or quotes
5. **Tables**: Ideal for comparisons, data, schedules, or structured information

## üêõ Troubleshooting

### Table Issues
- **Table not inserting**: Ensure cursor is in the editor, not in the title
- **Right-click menu not showing**: Make sure you're clicking directly on a table cell
- **Table formatting lost**: Tables are preserved in PDF export but not in text export

### General Issues
- **Auto-save not working**: Check your internet connection
- **AI features unavailable**: Ensure you're logged in and have an active session
- **Formatting not applying**: Try refreshing the page or clearing browser cache
