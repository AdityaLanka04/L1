# Backend Structure

## Core Files
- `main.py` - FastAPI application entry point
- `database.py` - Database configuration and session management
- `models.py` - SQLAlchemy database models
- `auth.py` - Authentication and authorization

## API Modules
- `adaptive_learning_api.py` - Adaptive learning endpoints
- `flashcard_api_minimal.py` - Flashcard CRUD operations
- `notes_api_enhanced.py` - Notes management
- `question_bank_enhanced.py` - Question bank functionality

## AI & Processing
- `ai_chat_agent.py` - AI chat agent
- `ai_chat_integration.py` - Chat integration logic
- `ai_utils.py` - AI utility functions
- `ai_personality.py` - AI personality configuration
- `advanced_prompting.py` - Advanced prompt engineering
- `concept_classification_agent.py` - Concept classification

## Media Processing
- `media_processor_unified.py` - Unified media processing
- `media_manager.py` - Media file management
- `media_models.py` - Media-related models
- `pdf_processor.py` - PDF processing
- `audio_transcriber.py` - Audio transcription
- `ai_media_processor.py` - AI-powered media processing
- `comprehensive_slide_analyzer.py` - Slide analysis

## Learning Systems
- `adaptive_learning_engine.py` - Adaptive learning algorithms
- `neural_adaptation.py` - Neural network adaptation
- `gamification_system.py` - Gamification features
- `study_session_analyzer.py` - Study session analytics

## Services
- `storage_service.py` - File storage (local/cloud)
- `import_export_service.py` - Data import/export
- `websocket_manager.py` - WebSocket connections
- `proactive_ai_system.py` - Proactive AI notifications

## Utilities
- `math_processor.py` - Math expression processing
- `question_bank_models.py` - Question bank data models

## Directories
- `agents/` - AI agent implementations
- `ml/` - Machine learning models
- `models/` - Additional model definitions
- `uploads/` - Uploaded files storage
- `attachments/` - File attachments
- `chroma_db/` - Vector database storage
